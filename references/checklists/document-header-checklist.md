# 文档头自检清单

用于在以下场景快速自检文档头元信息是否仍与正文用途一致：
- 当前阶段主文档首次落盘后
- 当前阶段确认后
- review / 验收结论形成后
- 固定入口文档切换到新轮次后
- 恢复项目并修正文档自洽时

---

## 自检项
- [ ] 文档名称是否与当前文件用途一致
- [ ] 当前状态是否与当前阶段结论一致
- [ ] 所属阶段是否正确
- [ ] 所属项目是否正确
- [ ] 所属功能 / 子功能是否仍停留在旧轮次
- [ ] 上游文档是否已切到当前有效文档链
- [ ] 创建时间 / 最后更新时间是否合理
- [ ] 若该文档是固定入口文档（如 `review.md` / `sdd-status.md`），头部是否已同步切换到当前轮次用途

---

## 使用提示
一句话判断：
> 如果正文已经是当前轮次，文档头也必须一起变成当前轮次；不允许正文是新的，头还是旧的。
